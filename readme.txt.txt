Campus Security Staff Management System
---------------------------------------

How to run the project:

1. Download the XAMPP control Panel app.

2. Import the database:
   - Open phpMyAdmin
   - Create a new database: campus_security
   - Click "Import" and choose campus_security.sql

3. Move the folder:
   - Copy 'campus-security' into C:\xampp\htdocs\

4. Launch XAMPP:
   - Start Apache and MySQL

5. Open the browser:
   - Go to http://localhost/campus-security/index.html
